# Coffee Shop Sales Visualization

This project uses Matplotlib and Seaborn to visualize coffee shop sales data.

## Visualizations Included:
✔ Monthly Sales Trend  
✔ Category-wise Sales  
✔ Time Period Sales  
✔ Top 10 Best-Selling Products  
✔ Sales Histogram  
✔ Category-wise Boxplot  

The notebook explains each graph with clean, readable code.
